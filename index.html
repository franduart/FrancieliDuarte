Site para ONG de Adoção de Cães

Ajudei a ONG Alma viralata a receber doações e facilitar a adoção dos seus cães. Site responsivo, com painel de cadastro de animais.

Tecnologias: HTML, CSS, JS, Firebase

Projeto primeFlix

Projeto desenvolvido em React, onde consumi uma API de filmes para exibir um catálogo dinâmico. Os dados são carregados em tempo real, com informações como título, pôster, sinopse e avaliação dos filmes.

React, CSS Modules e The Movie Database API (TMDb)
Ver projeto GitHub

Site de Hamburgueria com Pedido via WhatsApp

Criei um site moderno e responsivo para uma hamburgueria local, onde o cliente pode montar seu pedido diretamente pelo navegador e finalizar pelo WhatsApp com apenas um clique. O site permite escolher lanches, acompanhamentos e bebidas, e gera uma mensagem automática com o resumo do pedido.
Tudo foi feito com HTML, JavaScript puro e estilizado com Tailwind CSS para um visual limpo e atrativo.

HTML, JavaScript, Tailwind CSS, WhatsApp API
Ver projeto GitHub
